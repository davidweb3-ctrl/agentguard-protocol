# Demo Scripts

Colosseum recommends preparing two different videos:

- a pitch video that explains the why
- a technical demo video that explains the how

Use these scripts:

- [Pitch Video Script](PITCH_VIDEO_SCRIPT.md): primary 3-minute founder-style submission video.
- [Technical Demo Script](TECHNICAL_DEMO_SCRIPT.md): 2-3 minute implementation walkthrough.
- [Recording Checklist](RECORDING_CHECKLIST.md): terminal layout, shot list, and final upload checks.

## Suggested Submission Order

1. Record the pitch video first. It is usually the first thing judges review.
2. Record the technical demo second. It should prove the implementation without repeating the pitch.
3. Keep both videos in English.
4. Keep access permissions public or unlisted, but accessible to judges.
