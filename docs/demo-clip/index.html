1. 402 challenge The paid API asks the agent for a Solana payment proof.
2. Guarded payment The agent can pay only through owner-defined policy.
3. Receipt unlock The API unlocks only after a decoded PaymentReceipt matches the challenge.
4. Over-limit rejection Rejected payments do not move funds.
